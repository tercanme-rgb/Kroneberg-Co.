# Starter vs PRO Mapping

## Starter visible
- meta
- dashboard
- scoring
- market_research.market_overview
- community_signals.summary
- community_signals.pain_points
- competitor_insights.summary
- verdict

## Starter locked
- market_research.market_size
- competitor_insights.competitor_table
- competitor_insights.pricing_landscape
- strategy_frameworks
- business_model
- case_examples
- roadmap_30d
- visuals.trend_chart
- PDF export

## PRO visible
Everything in the schema, unless explicitly hidden by business logic.

## Practical rule
Starter should answer:
- “Is this interesting?”

PRO should answer:
- “Do I know how to enter this market?”
