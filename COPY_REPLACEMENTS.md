# NicheLab copy replacements

Use these replacements across `index.html`, `dashboard.html`, pricing blocks, report CTAs, and upgrade prompts.

## Global replacements
- `Full 13-section report` -> `Detailed report`
- `13-section report` -> `Detailed report`
- `Full Report` -> `Open report`
- `Open full brief` -> `Open report`
- `Read today’s report free` -> `Read today’s analysis`
- `Unlock full 13-section report` -> `Unlock detailed report`
- `Access full 13-section reports` -> `Access detailed reports`

## Recommended button copy
- Featured CTA -> `Open report →`
- Leaderboard CTA -> `Open report`
- Upgrade CTA -> `Upgrade for deeper analysis`
- PDF CTA -> `PDF export available on Pro`

## Plan copy
### Starter
`Starter includes curated opportunity briefs, live report access, and core market snapshots.`

### Pro
`Pro unlocks detailed reports, richer report depth, downloadable exports, and expanded analysis layers.`

## Suggested terminology
- Starter -> `Opportunity brief`
- Pro -> `Detailed report`
