# Report Generation Pipeline

## 1. Input layer
Collect:
- title
- subtitle
- category
- geo
- short thesis
- audience

## 2. Scoring layer
Generate:
- market_score
- revenue_score
- competition_score
- difficulty_score
- speed_to_mvp_score
- founder_fit_score
- signal_strength_score

This can be heuristic at first.

## 3. Research layer
Populate:
- market_research
- community_signals
- competitor_insights
- strategy_frameworks
- business_model
- founder_fit
- risk_matrix
- case_examples
- roadmap_30d

## 4. Visual layer
Generate configs for:
- trend_chart
- score_bars
- risk_visual
- extra tables/cards

## 5. Access layer
Set:
- tier_visibility
- starter_view
- pro_view

## 6. Rendering layer
Use the same JSON to produce:
- dashboard card
- starter report
- pro report
- PDF export
